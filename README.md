## Installation
```
pip install backtest
```


## Usage

The following will execute the backtest after loading configuration file, `config.json` in same directory.
```python
python -m backtest
````

Sample content of `config.json`
```json
{
    "market": "korea",
    "strategy": "dummy",
    "ticks_dir": "ticks",
    "ledger_dir": "ledger",
    "cash": 1000000
}
```
